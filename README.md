# L.E.A.D. Ministry – Next.js Starter

**Stack**: Next.js (App Router) + Tailwind + TypeScript  
**Brand**: Teal `#008080`, Brown `#5A3E2B`, Beige `#F9F6F1` · Font: Montserrat

## Quick Start
```bash
npm install
npm run dev
# visit http://localhost:3000
```

## Tailwind
Already configured in `tailwind.config.js` and `app/globals.css`.

## Deploy
- Create a new project on Vercel and import this repo.
- Set framework: Next.js. Build command: `next build`. Output: `.next` (defaults).
- Add your domain when ready (e.g., leadministry.org).

## Next steps
- Replace placeholder text with your content.
- Hook up forms to Formspree/Tally or your backend.
- Add dynamic pages under `app/(site)/*`.

## Forms (Formspree)
- Replace `your_prayer_form_id` and `your_application_form_id` in `app/page.tsx` with your actual Formspree IDs.
- Create forms at https://formspree.io/ → copy the endpoint ID.
